# BlazonCalc

This version uses XcodeGen so Codemagic generates a clean modern `.xcodeproj` during the build.

Codemagic:
- Project path: `.`
- Workflow: `Build BlazonCalc`

Do not manually select an Xcode project if Codemagic uses the YAML workflow. The build generates `BlazonCalc.xcodeproj` automatically.
